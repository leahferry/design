Code & Craft

Tension in Space

Code & Craft is an interactive learning installation developed in collaboration with MaKenna Daniels and Tori Puff that bridges computer science and knitting to explore the tension between structure and social expectation. Guided by a language of push and pull, knit patterns and code generate a responsive framework that challenges gendered stereotypes while encouraging collaboration between digital and tactile making. A series of interactive stations support experimentation, skill-building, and collective learning across ages and experience levels. Through this interdisciplinary approach, the installation positions technology as a tool for connection, creating a shared learning landscape rooted in creativity, accessibility, and exchange.
